Makes the meta box in pages/posts sticky when you scroll.

== Description ==

This is a simple plugin that makes the meta box sticky on your pages/posts/custom posts when the content on the page is long.

== Installation ==

Installation

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress

Usage

1. Head to a long page/post/custom post and scroll to the bottom.
